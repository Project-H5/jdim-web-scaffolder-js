# 咚咚H5脚手架-TypeScript版
## 业务功能
1. **暗黑模式**
   - 预置暗黑适配相关接口
   - 预置暗黑适配相关资源
2. **烛龙监控**
   - 性能监控（自动采集）
   - 请求监控（自动采集）
   - 白屏监控（自动采集）
   - JS错误（自动采集）
   - 静态资源监控（自动采集）
   - 业务监控相关接口
3. **圆角适配**
   - 圆角大小获取接口
## 配置功能
1. **React 环境**
   - 预装最新版本的 React 和 ReactDOM
   - 支持所有现代 React 特性，如 Hooks

2. **Webpack 构建系统**
   - 自动配置的生产和开发构建过程
   - 代码分割和打包优化

3. **Babel 转译**
   - 支持最新的 JavaScript 特性
   - JSX 转译
   - 可配置的浏览器兼容性

4. **CSS 支持**
   - 支持 CSS 模块
   - 支持 Sass（需要安装 node-sass）
   - PostCSS 处理（如自动添加浏览器前缀）

5. **开发服务器**
   - 内置的开发服务器，支持热模块替换（HMR）
   - 自动刷新浏览器

6. **代码优化**
   - 生产环境下的代码压缩和优化
   - 移除无用代码

7. **ESLint**
   - 预配置的 ESLint 设置
   - 在开发过程中提供实时的代码质量反馈

8. **测试环境**
   - Jest 测试框架
   - 内置的测试运行器

9. **环境变量**
   - 内置支持 .env 文件
   - 区分开发和生产环境的环境变量

10. **PWA 支持**
    - 可选的 Progressive Web App 功能
    - 包括 Service Worker 和 Web App Manifest

11. **文件处理**
    - 支持图片、字体等静态资源的导入
    - 文件指纹（文件名哈希）用于缓存破坏

12. **错误处理**
    - 开发环境下的错误覆盖层
    - 生产环境下的错误报告

13. **代码分割**
    - 支持动态 import() 语法
    - 自动生成代码分割点
